-   Font sizes:
    -   General: 12px
    -   Title: 16px
    -   Total count: 32px

## Live Demo

Try out a live demo [here](https://example.com).

## Getting Started

Execute the following commands to build and run the example.

```bash
$ yarn
$ yarn start
```

Now point your browser to http://localhost:3000/.
